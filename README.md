# eleventy-dts

Types for the default export from `@11ty/eleventy`


## Install
```shell
$ npm install eleventy-dts
```

## Add to `tsconfig.json`
```json
{
	"compilerOptions":{
		// ...
	},
	"files": [
		// ...
		"node_modules/eleventy-dts/src/index.d.ts"
	]
}

```

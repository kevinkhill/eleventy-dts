# eleventy-dts

Types for the default export from `@11ty/eleventy`


## Install
download and copy into your project

## Add to `tsconfig.json`
```json
{
	"compilerOptions":{
		// ...
	},
	"files": [
		"eleventy-dts/src/index.d.ts"
		// ...
	]
}

```
